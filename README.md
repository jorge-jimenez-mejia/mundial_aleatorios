# 🏆 FIFA WORLD CUP 2026 DRAW

Python script for a live 2026 FIFA World Cup sweepstakes draw.

## Features

* 12 participants
* 48 World Cup teams
* 4 teams per participant
* 1 guaranteed elite team per person
* Fully randomized draw
* Interactive live reveal system
* Validation checks:

  * No duplicate teams
  * All 48 teams assigned
  * Everyone receives 4 teams

## How It Works

* Elite teams and other teams are shuffled randomly
* Participant order is shuffled randomly
* Each participant receives:

  * 3 random teams
  * 1 elite team

## Run

```bash
python world_cup_draw_live.py
```
